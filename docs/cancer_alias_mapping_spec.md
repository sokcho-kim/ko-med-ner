# 암 별칭 → KCD 코드 매핑 명세

> 작성일: 2026-01-07
> 목적: NER Entity Linking에서 사용하는 암 별칭 매핑의 근거 문서화

---

## 1. 개요

### 1.1 문제 정의

KCD(한국표준질병사인분류)에서 암은 정식 의학 명칭으로 등록되어 있음:
- 예: "결장의 악성 신생물" (C18), "상세불명의 결장의 악성 신생물" (C18.9)

그러나 일상적으로 사용되는 통용 명칭은 다름:
- 예: "대장암", "폐암", "위암"

→ **통용 명칭 → KCD 코드 매핑 필요**

### 1.2 매핑 원칙

| 원칙 | 설명 |
|------|------|
| 상세불명(.9) 코드 사용 | 부위 특정 없이 통용되는 명칭은 "상세불명" 코드에 매핑 |
| 공식 분류 체계 준수 | KCD-9 (2025년 기준 최신) 기준 |
| 점(.) 생략 허용 | C18.9 = C189 (코드 정규화) |
